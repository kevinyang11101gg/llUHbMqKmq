# 项目简介

本项目是一款基于Python和Django框架构建的花卉商城系统，附带1.2万字文档，提供源码下载。

## 核心功能
- 登录注册：用户和管理员可通过登录注册功能访问系统。
- 花卉信息：展示各类花卉信息，包括图片、描述和价格。
- 商城资讯：发布商城最新动态和活动信息。
- 花卉信息详情：提供花卉的详细信息，包括种植技巧和养护方法。
- 用户管理：管理员可管理用户信息，包括添加、修改和删除。
- 自助订花管理：用户可在线下单购买花卉，并查看订单状态。
- 花卉信息管理：管理员可添加、修改和删除花卉信息。
- 订单管理：管理员可查看和管理订单信息。
- 系统管理：管理员可进行系统设置和配置。

## 适用场景
- 花卉零售商
- 花卉电商企业
- 花卉爱好者社区

## 技术特点
- 使用Python语言和Django框架，确保系统稳定性和安全性。
- 采用MySQL数据库存储数据，保证数据持久性和完整性。
- 前端使用Vue.js框架，提供良好的用户体验。
- 代码可运行，提供详细的配置步骤，易于上手。
